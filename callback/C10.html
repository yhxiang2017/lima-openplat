1. 续保成功通知

平台在保险公司有保单续保成功后，将通过此接口将续保出单信息通知合作方

1.1. 请求入参

参考保单数据回传renewalInfo节点

1.2. 返回参数

字段 名称 类型 大小 必传 备注
messageHead 响应信息头 object 32 M
messageHead.channelId 商户号 string M
messageHead.messageCode 响应码 string M 0: 处理成功，非0处理失败，会触发重试
messageHead.messageDesc 响应描述 string M
messageHead.requestType 请求类型 string M
messageHead.UUID 唯一标识 string M
messageHead.version 版本 string M

1.3. 返回示例

{
    "messageHead":
    {
        "channelId": "1308911097474727936",
        "messageCode": "0",
        "messageDesc": "接口成功执行",
        "requestType": "C05",
        "UUID": "0000646d222657984c429d350865555f439c",
        "version": "1.0"
    },
    "response":
    {}
}

1. 保单详情查询

可通过该接口查询到某一条保单的所有详细信息，可作为保单同步的补偿机制接口

1.1. 请求方法

POST

1.2. 请求路径

/rainforest/service/gateway/v6/tenant/policyDetail

1.3. 请求入参

字段 名称 类型 大小 必传 备注
policyId 保单id string C 保单id

1.4. 入参示例

{
    "messageHead":
    {
        "version": "",
        "requestType": "R27",
        "uuid": "",
        "sendTime": "2023-03-20"
    },
    "request":
    {
        "policyId": ""
    }
}

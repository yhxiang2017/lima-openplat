1. 回执数据查询

可通过该接口查询回执清单, 查询区间必须在10天内

1.1. 请求方法

POST

1.2. 请求路径

/rainforest/service/gateway/v6/tenant/receiptList

1.3. 请求入参

字段 名称 类型 大小 必传 备注
startTime 查询开始时间 string C yyyy-MM-dd HH:mm:ss
endTime 查询结束时间 string C yyyy-MM-dd HH:mm:ss
spuId 产品id string C
policyIds 此字段传值时，开始及结束时间非必填，反则必传，最多3000 list C

1.4. 入参示例

{
    "messageHead":
    {
        "version": "",
        "requestType": "R30",
        "uuid": "",
        "sendTime": "2023-03-20"
    },
    "request":
    {
        "startTime": "2023-03-20 00:00:00",
        "endTime": "2023-03-20 23:59:59",
        "spuId": "2023-03-20 23:59:59",
        "policyIds": ["11111","22222"]
    }
}

1.5. 返回参数

字段 名称 类型 大小 必传 备注
data 数据集合 list M
data[0].policyId 保单id string M
data[0].receiptStatus 回执状态 string M 见回执回访状态
data[0].receiptTime 回执时间 string M yyyy-MM-dd HH:mm:ss
data[0].receiptStatusDesc 回执状态描述 string M
fileData 查询失败的数据集合 list C 根据保单id查询时才会有
fileData.policyId 保单id string C
fileData.failDesc 失败信息描述 string C

1.6. 返回示例

{
    "messageHead":
    {

        "messageDesc": "成功"
    },
    "response":

            {
                "policyId": "12121",

            }
        ],

            {
                "policyId":"",

            }
        ]
    }
}

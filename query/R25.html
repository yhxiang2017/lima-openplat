1. 保单状态查询

可通过该接口查询到某一保单当前在平台的保单状态

1.1. 请求方法

POST

1.2. 请求路径

/rainforest/service/gateway/v6/tenant/policyStatus

1.3. 请求入参

字段 名称 类型 大小 必传 备注
policyId 保单id string C 根据保单号查询状态，不传，查所有前一天发生过状态变更的

1.4. 入参示例

{
    "messageHead":
    {
        "version": "",
        "requestType": "R25",
        "uuid": "",
        "sendTime": "2023-03-20"
    },
    "request":
    {
        "policyId": ""
    }
}

1.5. 返回参数

字段 名称 类型 大小 必传 备注
data 数据集合 list M
data[0].policyId 平台订单号 string M
data[0].policyStatus 支付链接 string M
data[0].updatedTime 更新时间 date M 格式为: yyyy-MMdd HH:mm:ss

1.6. 返回示例

{
    "messageHead":
    {
        "requestType": "R25",
        "version": "",
        "UUID": "",
        "messageCode": "",
        "messageDesc": "",
        "channelId": "",
        "sendTime": "2023-03-20",
        "asyn": ""
    },
    "response":
    {
        "data":
        [
            {
                "policyId": "",
                "policyStatus": "",
                "updatedTime": ""
            }
        ]
    }
}

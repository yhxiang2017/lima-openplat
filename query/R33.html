1. 订单详情查询

可通过该接口查询续期清单, 查询区间必须在10天内

1.1. 请求方法

POST

1.2. 请求路径

/rainforest/service/gateway/v6/tenant/proposalDetail

2. 请求入参

字段 名称 类型 大小 必传 备注
proposalId 订单id string M

2.1. 入参示例

{
    "messageHead":
    {
        "version": "",
        "requestType": "R33",
        "uuid": "",
        "sendTime": "2023-03-20"
    },
    "request":
    {
        "proposalId": ""
    }
}

2.2. 返回参数

字段 名称 类型 必传 备注
order 数据集合 object M
order.orderId 平台订单号 string M
order.parentOrderId 批次父级订单号 string C 团单或者多被保人多单的情 况必传，以此标志当前订单 属于哪一批次的子订单
order.policyStatus 保单状态 string M 出单回调即为已承保状态
order.productId 平台产品ID string C
order.productName 产品名称 string C
order.policyId 保单号 string M
order.bizTag 商务业务标签 string C
order.totalPremium 总保费 string M 以分为单位
order.beginDate 保险开始时间 string C “yyyy-MM-dd HH:mm:ss”
order.endDate 保险结束时间 string C “yyyy-MM-dd HH:mm:ss”
order.payType 缴费方式 string C Y：年、M：月、Q：季 度、SP：一次性缴清
order.payPeriod 缴费期间 string C Y5 、Y10 5 年 和 10 年缴， SP: 趸交， A30: 交至30 周岁
order.insPeriod 保险期间 string C Y10 十年、D1 一 天A60 保到 60 岁、O 终生
order.amount 保额 string C 以元为单位，当产品为短险 产品时 amount 字段不 回 传
order.payTime 支付时间 string C “yyyy-MM-dd HH:mm:ss”
order.productCategory 产品分类名称 string C
order.agencyId 平台代理人ID string C
order.insureOrgId 保险公司ID string C
order.insSource 出单来源 string C coop-合作方平台； lima平台
order.insureOrgName 保险公司名称 string C
order.skuList 方案信息 list M
order.skuList.skuId 方案编号 string M
order.skuList.skuName 方案名称 string C

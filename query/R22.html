1. 获取电子保单

合作方根据保单号等信息在电子保单生成后可调用此接口获取电子保单链接。
对于出单成功的订单，开发者可调用此服务完成下载电子保单的操作。
提示：此接口不要在出单回调的同时同步调用，可在出单成功后异步调用或者在用户请求时调用。因：部分保司电子保单生成时间为5-30分钟。

1.1. 请求方法

POST

1.2. 请求路径

/rainforest/service/gateway/v6/tenant/policyUrl

1.3. 请求入参

字段 名称 类型 大小 必传 备注
coopOrderId 合作方订单号 string C cps对接时非必传
orderId 平台订单号 string M
policyId 保单号 string M

1.4. 入参示例

{
    "messageHead":
    {
        "version": "",
        "requestType": "R22",
        "uuid": "",
        "sendTime": "2023-03-20"
    },
    "request":
    {
        "coopOrderId": "",
        "orderId": "",
        "policyId": ""
    }
}

1.5. 返回参数

字段 名称 类型 大小 必传 备注
policyId 保单号 string 32 M
epolicyurl 电子保单下载地址 string M

1.6. 返回示例

{
    "messageHead":
    {
        "requestType": "R22",
        "version": "",
        "UUID": "",
        "messageCode": "",
        "messageDesc": "",
        "channelId": "",
        "sendTime": "2023-03-20",
        "asyn": ""
    },
    "response":
    {
        "policyId": "",
        "epolicyUrl": ""
    }
}
